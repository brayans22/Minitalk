# ➡︎ Server & Client (Minitalk) ⬅︎

### Summary
This project consists of communication between a server and client through unix signals.

### Usage
1) Compile the code
   ```makefile
   make
   ```
2) Execute server
   ```bash
   ./server
   ```
3) Execute client
   ```c
   ./client <PID> "Message to send to the server"
   ```

### Info
◼️ Language: C
<br>
◼️ Compile: makefile
